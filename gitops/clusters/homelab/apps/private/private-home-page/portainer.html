Docker Registry — Publish Image

Registry192.168.1.7:5000 UI192.168.1.7:8282

1 — Allow insecure registry (once per Docker host)

# macOS: ~/.docker/daemon.json
# Linux: /etc/docker/daemon.json
{
  "insecure-registries": ["192.168.1.7:5000"]
}
# macOS: restart Docker Desktop
# Linux: sudo systemctl restart docker

2 — Tag your image

docker tag <image>[:tag] 192.168.1.7:5000/<name>[:tag]

3 — Push

docker push 192.168.1.7:5000/<name>[:tag]

Pull

docker pull 192.168.1.7:5000/<name>[:tag]
